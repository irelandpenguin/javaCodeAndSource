if条件语句
1. 有两个整型变量x=10, y=13。利用if条件语句判断这两个整型变量的大小，打印输出较大的那个变量值。
2. 有一个整型变量p，任意给它一个初值。利用if条件语句判断变量p是奇数还是偶数。
3. 看程序写结果：请自己独立分析，先不要编译运行。
int x = 1,y = 1;
if(x++==2 && ++y==2) {
	x =7;
}
System.out.println("x="+x+",y="+y);
